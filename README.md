# Autumn-framework
Autumn framework is a minimal Spring Framework.

Components of Autumn Framework:

- (In progress)Autumn-context: Core IoC container that support annotation-based injection;
- (not started yet)Autumn-aop: AOP support for annotation-based subclassing proxy;
- (not started yet)Autumn-jdbc: Provides JdbcTemplate and declarative transaction management;
- (not started yet)Autumn-web: Support web application with Servlet 6.0;
- (not started yet)Autumn-boot: Run Autumn application simple and quick like Spring Boot.
